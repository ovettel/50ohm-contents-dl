Energie hat die Einheit $\unit{\joule}$ (für Joule).  
Ein Joule ($\qty{1}{\joule}$) entspricht einem Watt ($\qty{1}{\watt}$), das über eine Dauer von einer Sekunde ($\qty{1}{\second}$) wirkt: $\qty{1}{\joule} = \qty{1}{\watt\second}$.  
Damit entspricht $\qty{1}{\watt\hour} = \qty{3600}{\joule}$.

Tipp: An die Stromrechnung denken, dort wird der Stromverbrauch üblicherweise auch in kWh angegeben ($\qty{1}{\kilo\watt\hour} = \qty{1000\cdot3600}{\joule}$).  

Ausschlussverfahren: Volt ist eine Spannung, Watt ist eine Leistung (also Energie pro Zeit).
